# Sales Insight PowerBI

## Overview
This project demonstrates data visualization and business intelligence using **Microsoft Power BI**.  
It transforms raw data into interactive dashboards and insightful reports to support decision-making.

## Dataset
- Source: `Power BI - Final Project.xlsx`
- Contains raw transactional/business data used for analysis.

## Report
- Power BI file: `Power BI - Final Project.pbix`
- Includes interactive dashboards for:
  - Sales performance
  - Customer analysis
  - Time-based trends
  - KPIs

## How to Use
1. Download this repository.
2. Open the `.pbix` file with **Microsoft Power BI Desktop**.
3. Connect to the dataset if needed.
4. Explore the interactive dashboard.
